The system calls that relate to the dynamic linking here are open, fstat,
mmap, read, and close. "open" is for opening the files that the libraries at
their designated locations; "fstat" is for getting the file information
(possibly the file size as well), "mmap" is for allocating space (particularly
for the contents in the library to place in the object file); "read" is for
actual transfer of the contents; "close" is for closing the file after the
program is done with it.
